## ✨ Key Features

- **Dynamic Hero Section:** A captivating introduction with micro-animations that immediately grabs attention.
- **Services Showcase:** Elegantly highlights the core consulting services offered by DTW Consult.
- **Interactive About Us:** A deep dive into the company's ethos, presented with smooth scroll-linked animations.
- **Contact & Outreach:** A fully functional, beautifully designed contact form to generate leads.
- **Mobile-First Design:** Flawless responsiveness across all devices, ensuring a perfect experience on smartphones, tablets, and desktops.
- **Intelligent Backend Integrations:** Primed for AI interactions and backend services using Express and Google GenAI.

## 🛠️ Technology Stack

This project is built on a robust and bleeding-edge stack:

- **Frontend Framework:** [React 19](https://react.dev/) for a reactive, component-based UI.
- **Build Tool:** [Vite 6](https://vitejs.dev/) for instantaneous HMR and optimized builds.
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) for rapid, utility-first styling with custom design tokens.
- **Animations:** [Framer Motion](https://www.framer.com/motion/) for fluid, physics-based micro-interactions.
- **Icons:** [Lucide React](https://lucide.dev/) for crisp, scalable vector icons.
- **Backend/AI:** Express.js and `@google/genai` for advanced server-side capabilities.

## 🚀 Getting Started

Want to run this masterpiece locally? Follow these simple steps:

### Prerequisites

Make sure you have Node.js installed (v18+ recommended).

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ederaefe/DTWCL.git
   cd DTWCL
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Copy the `.env.example` file to `.env` and fill in any required keys.
   ```bash
   cp .env.example .env
   ```

4. **Spin up the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to `http://localhost:3000` to see the magic happen!
